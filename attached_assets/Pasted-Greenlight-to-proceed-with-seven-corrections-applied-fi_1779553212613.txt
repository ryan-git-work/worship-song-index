# Correction 4 — Add a `formatTimeSignature` helper for slash-vs-hyphen handling

The URL slug for 6/8 is `6-8` (hyphen, because slashes break URL paths). Display should be `6/8`. The proposed code uses `signature` (the raw slug) in both URL and name fields, which prints things like "Time Signature: 6-8" instead of "Time Signature: 6/8".

Add a helper (wherever utilities live):
function formatTimeSignature(slug) {
// "6-8" → "6/8", "12-8" → "12/8", "3-4" → "3/4"
return slug.replace("-", "/");
}

- **URL fields** (`url`, `item` inside breadcrumb ListItems, canonical) → use the raw slug `${signature}`
- **Display fields** (`name`, `description`, breadcrumb `name`) → use `${formatTimeSignature(signature)}`
